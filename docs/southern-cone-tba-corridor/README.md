# Southern Cone Human Trafficking — TBA Corridor

**Author:** Daniela Rutman (DCR)  
**Issued:** May 2026  
**Status:** In Progress  
**Classification:** Unclassified // TLP:CLEAR  

## Overview

Open-source geospatial and documentary assessment of human trafficking infrastructure across the Tri-Border Area (TBA) corridor — Argentina, Brazil, and Paraguay.

## Sites Assessed

| Site | Location | Modality |
|------|----------|----------|
| Alpha | Foz do Iguaçu / Lake Itaipú · BR/PY | Water crossing — informal dock |
| Bravo | Ponta Porã / P.J. Caballero · BR/PY | Open dry border — no physical barrier |
| Charlie | Posadas / Encarnación · AR/PY | River crossing — clandestine access |
